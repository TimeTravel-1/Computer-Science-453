Usage

Use the "T" key to start the Sierpinski Triangle.
Use the "P" key to start the Parametric Spiral.
Use the "S" key to start the Squares and Diamonds. 
Use the “K” key to start the Koch Snowflake.
Use the “F” key to start the Barnsley Fern.
Use the keys "1-8" inclusive to switch between the iteration level of each scene.

Platform and Compiler

I have tested this on my Mac Operating System.
To compile and run the program, please type the following:

g++ boilerplate2.cpp -o A1 -framework OpenGL -lglfw3
./A1
